# Claude OS

Global Claude Code configuration, slash commands, and setup guide.

## Structure
- `global/CLAUDE.md` — global rules loaded in every session
- `global/settings.json` — hooks and permissions
- `global/commands/` — slash commands (/morning, /sync, /review, /audit, /genesis, /post-review)

## How to use
Files in `global/` mirror what lives in `~/.claude/`.
If you edit here, copy the file back to `~/.claude/` to apply it.
